# ¿Cómo iniciar el intérprete?
1.- Abre la carpeta "out".
2.- Dentro se encuentra la carpeta "artifacts".
3.- Abrimos el archivo LaScript.bat (Este ejecuta el archivo .jar).
4.- Ahora escriba la ruta del archivo .las y listo.
